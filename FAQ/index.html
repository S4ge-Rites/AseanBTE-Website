4 What is the Trial Builder Program?

The Trial Builder program allows visitors to build on the ASEAN server before becoming a builder in one of the build teams. Visitors are able to use buildings that they have built in the trial area as an application submission to join build teams on the BTE website.
- There are some easy houses for beginners to start with in Singapore. They all have decent 3D models for reference.
Just contact a Leader, Co-Leader or an Admin if you want to join the Trial Builder program, after that, do /warp trial in the server to be there. Outlines surrounding buildings are all ready for you to build in.
- Location: 1.302362, 103.886411 (search this on google map).
- Official builders can also build in this trial area aswell.

6 How do I teleport to a location on the server (for Java Edition)?

Use the command /tpll to teleport to any place in the world. Use a mapping application like Google Maps and/or Google Earthto find the coordinates of your desired location. For an example of the teleportation process, You can check out this GIF!
